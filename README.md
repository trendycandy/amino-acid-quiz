# 🧬 잔기 퀴즈 (Amino Acid Residue Quiz)

단백질을 이루는 20가지 표준 아미노산을, 그냥 외우는 게 아니라 **이름·특성·구조·무질서 성향**을 연결지어 익히도록 만든 인터랙티브 퀴즈입니다. 생화학·구조생물학을 공부하거나 가르치는 사람을 위한 가벼운 복습 도구예요.

## 무엇을 연습하나요

한 번에 한 잔기씩 문제가 나오고, 답을 고르면 그 아미노산의 **신상카드**(이름·코드·특성 분류·IDR 성향·골격 구조식·분자식·분자량·한 줄 노트)가 펼쳐집니다. 맞히든 틀리든 매 문제마다 작은 학습 카드를 한 장씩 쌓아가는 셈이에요.

## 네 가지 모드

| 모드 | 보여주는 것 | 맞히는 것 |
|------|-------------|-----------|
| **코드** | 1글자 코드 (예: `W`) | 전체 이름 또는 3글자 코드 |
| **특성** | 잔기 (코드 + 이름) | Lehninger 5군 분류 |
| **IDR 성향** | 잔기 (코드 + 이름) | 무질서/질서 촉진 또는 중립 |
| **구조** | 골격 구조식 | 어떤 아미노산인지 |

> 구조 모드는 Leu/Ile, Asn/Gln, Asp/Glu처럼 한 끗 차이 나는 잔기를 구분하는 눈을 길러줍니다.

## 다루는 분류 체계

- **특성 (Lehninger 5군)**
  - 비극성 지방족 — G, A, P, V, L, M, I
  - 방향족 — F, Y, W
  - 극성 비전하 — S, T, C, N, Q
  - 양전하(염기성) — K, H, R
  - 음전하(산성) — D, E
- **IDR(무질서) 성향 (Dunker/Williams 조성 기반)**
  - 무질서 촉진 — A, R, G, Q, S, E, K, P
  - 질서 촉진 — W, C, F, I, Y, V, L, N
  - 중립·문맥의존 — D, H, M, T

각 잔기 노트에는 IDR 연구에 쓸 만한 한 줄 메모(예: 프롤린의 α-나선 파괴, 글루타민의 polyQ 응집, 인산화·글리코실화 표적 등)가 함께 붙어 있어요.

## 데이터 정확성

- 20개 아미노산의 SMILES를 **RDKit으로 검증**해 분자식이 일치하는 값만 사용했습니다.
- 분자량도 RDKit 계산값(중성형 자유 아미노산 기준).
- 구조식은 SMILES로부터 SmilesDrawer가 그립니다.

## 플레이 방법

- 마우스로 보기를 누르거나, 키보드 `1`~`4`로 선택, `Enter`로 다음 문제
- 연속 정답 시 보너스 점수가 붙고, 우측 상단에 정확도가 집계됩니다
- 상단 탭으로 언제든 모드를 바꿀 수 있어요

## 한 줄 요약

> 20개 아미노산 × 4가지 관점 = 코드·특성·구조·무질서 성향을 한 번에 잡는 복습 게임.
